### Task 2: Interaction fixes (items 3, 6, 7)
- **Drag-release flash**: on release the Move event applies AFTER the frame renders (old pos w/o live offset → 1-frame flash). Fix: keep rendering the live drag offset on the release frame (clear DragState only after the event applies) OR apply desk events before render — pick the minimal correct fix in the established event flow; kittest: position on the release frame's render == drop position (expose via the existing testable seams if pixels aren't assertable — at minimum a regression test that the session pos updates in the same frame the drag clears).
- **Pinch zoom**: handle `egui::Event::Zoom(factor)` (trackpad pinch) in the desk AND map camera paths, multiplying zoom by the factor with the same pointer-anchoring + clamps; keep Ctrl+scroll at 1.0015^delta. Kittest: inject Event::Zoom → zoom changes, anchored, clamped.
- **Zoom controls**: status line gains [−] [100%] [+] [Fit] buttons (zoom step ×1.25 per click, 100% sets zoom=1.0 anchored at viewport center); a11y-labeled; work on desk and map; gated like other mouse paths.
